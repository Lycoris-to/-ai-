# 数据结构学习中心 🎓

> 从概念到实践，系统化掌握数据结构与算法

## 🌐 在线访问

👉 **https://lycoris-to.github.io/-ai-/**

## 📚 内容模块

- 📊 **线性结构** — 顺序表、链表、栈、队列
- 🌳 **树形结构** — 二叉树、BST、AVL、堆、哈夫曼树
- 🕸️ **图形结构** — 图存储、遍历、最短路径、最小生成树
- 🔍 **查找算法** — 二分查找、哈希表
- 📋 **排序算法** — 冒泡、选择、插入、快排、归并、堆排
- 🤖 **AI 代码评估** — 提交代码，AI 分析问题并修正
- 💬 **AI 学习助手** — 随时解答数据结构问题

## 🚀 本地运行

```bash
pip install -r requirements.txt
python server.py
```
